🧩 Pentomino Solutions Gallery
Bu proje, 6x6 grid üzerinde yerleştirilmiş pentomino çözümlerini görsel olarak incelemek için hazırlanmış basit ve mobil uyumlu bir web sayfasıdır.

🎯 Özellikler
6x6 grid için hazırlanmış 35 farklı çözüm
Her çözüm PNG görsel olarak gösterilir
Alttaki numaralara tıklayarak çözüm değiştirme
Mobil uyumlu tasarım 📱
Solda kullanılan pentomino parçalarının gösterimi

🧱 Kullanılan Parçalar
Projede şu pentomino parçaları kullanılmıştır:
I - L  - N - P - U - X - Y
